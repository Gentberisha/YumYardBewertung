# YumYardBewertung

Eine einfache Website zur Bewertung von YumYard-Essen.

## Funktionen

- Abgabe von Bewertungen durch Benutzer
- Anzeige von Fake-Bewertungen, die die Frische und Leckerkeit des Essens hervorheben
- Speicherung der Benutzer-Bewertungen im Browser (localStorage)

## Verwendung

1. Öffnen Sie `index.html` in einem Webbrowser.
2. Geben Sie Ihre Bewertung im Formular ein.
3. Sehen Sie sich alle Bewertungen an, einschließlich der vorgegebenen Fake-Bewertungen.

## Dateien

- `index.html`: Haupt-HTML-Datei
- `styles.css`: CSS-Stile
- `script.js`: JavaScript für die Funktionalität
